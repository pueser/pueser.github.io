# ◆Regular Expression



## 1) 매칭 패턴

|   패턴    | 의미                                                         |
| :-------: | ------------------------------------------------------------ |
|  a-zA-Z   | 영어 알파벳으로 범위 지정                                    |
| ㄱ-ㅎ가힣 | 한글 문자로 범위 지정                                        |
|    0-9    | 숫자 범위로 지정                                             |
|     .     | 모든 문자열(숫자, 한글, 영어, 특수기호, 공백 단, 줄바꿈은 안됨) |
|    \d     | 숫자                                                         |
|    \D     | 숫자가 아닌 것                                               |
|    \w     | 영어 알파벳, 숫자, 언더스코어(_)                             |
|    \W     | 영어 알파벳, 숫자, 언더스코어(_)가 아닌것                    |
|    \s     | space 공백                                                   |
|    \S     | space 공백이 아닌것                                          |
| \특수기호 | 특수기호                                                     |
